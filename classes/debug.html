Clase Debug

La clase Debug es una simple utilidad para depurar variables, objetos, matríces, etc. mostrando la información de salida a la pantalla. Esta funcionalidad no solucionará todas tus necesidades de deburación, pero es una manera simple para lanzar información rápidamente para comprobar cosas a la misma vez.

dump(*$args)

El método dump imprimirá en pantalla valores múltiples mixtos al navegador en una manera formateada y estructurada y que incluirá formación de seguimiento para que sepas de dónde viene.

Estático Sí
Parámetros Argumentos ilimitados mixtos
Devuelve nada
Ejemplo 
Debug::dump($stuff, 'junk', array('whatever'));

Por defecto las estructuras anidadas serán mostradas colapsadas para minimizar el impacto de salida en la estructura de la página. Puedes alterar este comportamiento y tenerlo expandido ajustando Debug::$js_toggle_open a true.

backtrace()

El método backtrace te muestra el archivo y la línea desde la que el método es llamado y una lista de todos los métodos previos.

Estático Sí
Parámetros Ninguno
Devuelve nada
Ejemplo 
Debug::backtrace();

classes()

Muestra una lista de clases que están definidas actualmente en PHP en tiempo de ejecución.

Estático Sí
Parámetros Ninguno
Devuelve nada
Ejemplo 
Debug::classes();

interfaces()

Muestra una lista de todas las clases de interfaz actualmente definidas en PHP en tiempo de ejecución.

Estático Sí
Parámetros Ninguno
Devuelve nada
Ejemplo 
Debug::interfaces();

Estático Sí
Parámetros Ninguno
Devuelve nada
Ejemplo

Parámetros Ninguno
Devuelve nada
