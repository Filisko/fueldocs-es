Clase Agent

La clase Agent te permite conseguir información acerca del tipo de navegador, versión, plataforma o sistema operativo, etc. basado en la cadena de Agente de Usuario del cliente.

Uso

accepts_charset($charset = 'utf-8')

Comprueba si el navegador actual acepta una codificación de carácteres específica.

Estático Sí
Parámetros
Parámetro Por defecto Descripción
$charset string Nombre del conjunto de carácteres
Devuelve boolean
Ejemplo 
// Comprueba si el navegador del usuario acepta iso-8859-1
if (Agent::accepts_charset('iso-8859-1'))
{
	echo "¡Sí, el navegador acepta este conjunto de carácteres!";
}

accepts_language($language = 'en')

Comprueba si el navegador actual acepta un código de lenguaje ISO específico.

Estático Sí
Parámetros
Parámetro Descripción
$language string Código de lenguaje ISO
Devuelve boolean
Ejemplo 
// Comprueba si el usuario del navegador acepta dialécto flamenco (variedad regional del idioma neerlandés)
if (Agent::accepts_language('nl_BE'))
{
	echo "Sí, el navegador acepta el dialécto flamenco!";
}

browser()

Devuelve el nombre normalizado del navegador desde la cadena del usuario de agente.

Estático Sí
Parámetros Ninguno
Devuelve string
Ejemplo 
// Proceso basado en el nombre del navegador
switch (Agent::browser())
{
	case 'Firefox':
		// haz algo específico aquí
		break;
	case 'IE':
		// haz algo específico aquí
		break;
	case 'Chrome':
		// haz algo específico aquí
		break;
	case 'Unknown':
		// haz algo específico aquí
		break;
	default:
		// estos no son todos los navegadores soportados...
		break;
}

platform()

Estático Sí
Parámetros Ninguno
Devuelve string
Ejemplo 
// Proceso basado en la plataforma del navegador
switch (Agent::platform())
{
	case 'Win95':
	case 'Win98':
	case 'WinNT':
	case 'WinME':
	case 'Win2000':
		// haz algo específico aquí
		break;
	case 'WinXP':
	case 'WinVista':
	case 'Win7':
		// haz algo específico aquí
		break;
	case 'Linux':
		// haz algo específico aquí
		break;
	case 'MacOSX':
	case 'MacPPC':
		// haz algo específico aquí
		break;
	case 'SunOS':
	case 'FreeBSD':
	case 'Debian':
	case 'HP-UX':
	case 'IRIX64':
		// haz algo específico aquí
		break;
	case 'unknown':
		// haz algo específico aquí
		break;
	default:
		// no están todas las plataformas soportadas...
		break;
}

version()

Devuelve la versión del navegador

Estático Sí
Parámetros Ninguno
Devuelve float
Ejemplo 
// Comprueba si una versión antigua de Internet Explorer se está usando
if (Agent::browser() == 'IE' and Agent::version() < 7)
{
	echo "¡Lo sentimos, no soportamos navegadores anticuados!";
}

charsets()

Devuelve una matríz con todos los conjuntos de carácteres aceptados por el navegador.

Estático Sí
Parámetros Ninguno
Devuelve array
Ejemplo 
// Comprueba si el navegador del usuario acepta iso-8859-1
$sets = Agent::charsets();
if (in_array('iso-8859-1', $sets))
{
	echo "¡Sí, el navegador acepta este conjunto de cáracteres!";
}

languages()

Devuelve una matríz con todos los lenguajes ISO soportados por el navegador.

Estático Sí
Parámetros Ninguno
Devuelve array
Ejemplo 
// Comprueba si el usuario del navegador acepta dialécto flamenco (variedad regional del idioma neerlandés)

$lang = Agent::languages();
if (in_array('nl_BE', $lang))
{
	echo "¡Sí, el navegador acepta el dialécto flamenco!";
}

properties()

Devuelve una matríz con todas las propiedades del navegador detectado.

Estático Sí
Parámetros Ninguno
Devuelve array
Ejemplo 
// Obtener las propiedades del navegador
$propiedades = Agent::properties();

/**
 * Estas son las propiedades soportadas, y sus valores por defecto
 *
 * $propiedades = array(
 * 	'browser'             => "unknown",
 * 	'version'             => 0,
 * 	'majorver'            => 0,
 * 	'minorver'            => 0,
 * 	'platform'            => 'unknown',
 * 	'alpha'               => false,
 * 	'beta'                => false,
 * 	'win16'               => false,
 * 	'win32'               => false,
 * 	'win64'               => false,
 * 	'frames'              => false,
 * 	'iframes'             => false,
 * 	'tables'              => false,
 * 	'cookies'             => false,
 * 	'backgroundsounds'    => false,
 * 	'javascript'          => false,
 * 	'vbscript'            => false,
 * 	'javaapplets'         => false,
 * 	'activexcontrols'     => false,
 * 	'isbanned'            => false,
 * 	'ismobiledevice'      => false,
 * 	'issyndicationreader' => false,
 * 	'crawler'             => false,
 * 	'cssversion'          => 0,
 * 	'aolversion'          => 0,
 * );
 *
 * Date cuenta que los booleanos indican si una funcionalidad específica es soportada.
 * No indica si está habilitado o no.
 */

property($property = null)

Devuelve una propiedad específica del navegador. Vea la lista de propiedades soportadas a través de properties().

Estático Sí
Parámetros
Parámetro Por defecto Descripción
$property string Nombre de la propiedad a devolver. ¡El nombre es sensible a las mayúsculas!
Devuelve mixed
Ejemplo 
// Comprueba si el navegador soporta cookies.
if (Agent::property('cookies') === true)
{
	echo "¡Sí, el navegador soporte cookies!";
}

is_mobiledevice()

Comprueba si el navegador usado está siendo ejecutado en un dispositivo móvil.

Estático Sí
Parámetros Ninguno
Devuelve boolean
Ejemplo 
// Carga una vista por plataforma específica
if (Agent::is_mobiledevice())
{
	$contenido = View::forge('movil/nombre_de_vista');
}
else
{
	$contenido = View::forge('estandar/nombre_de_vista');
}

is_robot()

Comprueba si el agente del navegador indica si es un robot o es un crawler.

Estático Sí
Parámetros
Devuelve boolean
Ejemplo 
// Carga una vista por crawler específico (nota: ¡esto no se recomienda!).
if (Agent::is_robot())
{
	$contenido = View::forge('robot/nombre_de_vista');
}
else
{
	$contenido = View::forge('estandar/nombre_de_vista');
}
