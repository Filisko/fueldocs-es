Clase Crypt

La clase Crypt permite cifrar o descifrar una cadena de texto. La clase Crypt también es usada internamente por ejemplo por la clase Session de Fuel.

Usa los métodos de cifrado y hasheo proporcionados por PHPSecLib, así que no es dependiente de que haya o no módulos externos disponibles, como mcrypt.

Usage

Crypt puede ser usado de dos maneras: a través del uso estático de la clase Crypt y a través del objeto Crypt devuelto por los métodos Crypt::forge() o Crypt::instance(). Esta sección cubre el uso estático que siempre funcionará con la instancia por defecto usando la configuración especificada en la configuración.

Usando el objeto Crypt, los métodos Crypt::instance() y Crypt::forge() están explicados en la sección avanzado.

encode($value, $key = false)

El método encode cifra un valor de tipo string, opcionalmente con una llave de cifrado personalizada.

Estático No
Parámetros
Parámetro Por defecto Descripción
$value requerido Cadena a ser codificada.
$key 
false
 Valor de la llave personalizada opcional a ser usada para codificar el valor especificado. Si es falso, el valor de configuración 'crypto_key' es usado.
Devuelve string
Ejemplo 
// codifica una variable con una llave personalizada
$value = Crypt::encode($value, 'R@nd0mK~Y');

decode($value, $key = false)

El método decode descifra un valor de tipo string, opcionalmente con una llave de cifrado personalizada.

Estático No
Parámetros
Parámetro Por defecto Descripción
$value requerido Cadena a ser decodificada.
$key 
false
 Valor de la llave personalizada opcional a ser usada para decodificar el valor especificado. Si es falso, el valor de configuración 'crypto_key' es usado.
Devuelve mezclado - Cadena de texto con el valor decodificado, o false si el valor no pudo ser decodificado.
Ejemplo 
// decodifica una variable con una llave personalizada
$value = Crypt::decode($value, 'R@nd0mK~Y');
