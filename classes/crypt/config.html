Clase Crypt

La clase Crypt permite cifrar o descifrar una cadena de texto. La clase Crypt también es usada internamente por ejemplo por la clase Session de Fuel.

Usa los métodos de cifrado y hasheo proporcionados por PHPSecLib, así que no es dependiente de que haya o no módulos externos disponibles, como mcrypt.

Configuración

La clase Crypt está configurada a través del archivo de configuración app/config/crypt.php. Que será generado y rellenado con valores aleatorios la primera vez que uses la clase Crypt o si una configuración requerida no es encontrada.

¡Date cuenta de que esto requerirá tener acceso de escritura a app/config/crypt.php! Si esto no es posible, ¡asegúrate de que todos los ajustes de configuración están establecidos!

Los siguientes ajustes de configuración pueden ser establecidos:

Parámetro Tipo Por defecto Descripción
crypto_key string n/a Valor aleatorio de cifrado usado en las rutas de cifrado. ¡Asegúrese de configurar esto a algo único y al azar!
crypto_iv string n/a Vector de inicialización de cifrado aleatorio usado en las rutas de cifrado. ¡Asegúrese de configurar esto a algo único y al azar!
crypto_hmac string n/a Valor aleatorio usado en las rutinas de Hash-based Message Authentication Code (HMAC). Asegúrese de configurar esto a algo único y al azar!

Si asignas llaves manualmente, date cuenta de que están codificados a través de base64 (base64_encoded), y la longitud debe ser múltiplo de 4 para que sea posible decodificarlo. ¡Asegúrate de que la longitud sea la correcta!
