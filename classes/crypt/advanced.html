Clase Crypt

La clase Crypt permite cifrar o descifrar una cadena de texto. La clase Crypt también es usada internamente por ejemplo por la clase Session de Fuel.

Usa los métodos de cifrado y hasheo proporcionados por PHPSecLib, así que no es dependiente de que haya o no módulos externos disponibles, como mcrypt.

Uso avanzado

La clase Crypt soporta el uso de instancias múltiples. Esto te puede resultar útil si requieres conjuntos de claves de cifrado múltiple.

Puedes usar los mismos métodos en una instancia Crypt como lo haría usando métodos estáticos. Para el método You can use the same methods on an crypt instance as you would when you the static methods. Para la definición del método consulte la página del uso de la clase.

forge($config = array())

El método forge te permite instanciar manualmente una instacia de Crypt.

Estático Sí
Parámetros
Parámetro Por defecto Descripción
$name 
null
 Nombre de la instancia solicitada. Si no existe, será forzada. Si no se establece ningún nombre, la instancia por defecto será devuelta.
$config 
array()
 Puedes pasar configuración personalizada cuando se forcejee una instancia Crypt nueva. La matríz de configuración tiene la misma estructura que el archivo de configuración de Crypt. La configuración personalizada será fusionada con la configuración por defecto documentada aquí y con la configuración de tu archivo de configuración crypt, por lo tanto solo tienes que establecer los valores que quieras cambiar.
Devuelve object - El objeto Crypt instanciado.
Ejemplo 
// instancia un objeto Crypt con una clave personalizada
$crypt = \Crypt::forge(array(
	'crypto_key' => 'kj$Huhliuh7p*UHHhdas6%#@',
));

// y codifica alguna información con ell
$encoded = $crypt->encode('This is a very secure piece of data!');

instance($name = null)

El método instance te permite cargar una instancia con nombre de la clase Crypt.

Estático Sí
Parámetros
Parámetro Por defecto Descripción
$name 
null
 Nombre de la instancia solicitada. Si no existe, será forzada. Si no se establece ningún nombre, la instancia por defecto será devuelta.
Devuelve mezcla - El nombre de la instancia Crypt o falso si la instancia no existe.
Ejemplo 
// instancia un objeto Crypt con nombre con una clave personalizada
$crypt = \Crypt::instance('custom', array('crypto_key' => 'kj$Huhliuh7p*UHHhdas6%#@'));

// y codifica alguna información con ello
$encoded = $crypt->encode('This is a very secure piece of data!');

Los métodos estáticos de la clase Crypt también usan la instancia por defecto. Recuerda esto si alteras la configuración de la instancia por defecto.
