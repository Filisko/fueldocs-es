Clase Autoloader

La clase Autoloader contiene la función de autocarga de Fuel. También provee métodos para interactuar con el proceso de autocarga.

add_namespace($namespace, $path, $psr = false)

El método add_namespace añade un espacio de nombres y una ruta.

Estático Sí
Parámetros
Parámetro Por defecto Descripción
$namespace requerido el espacio de nombres
$path requerido ruta al espacio de nombres
$psr 
false
 establecer a true para indicar sigue el estándar de espacio de nombres PSR-0.
Devuelve Nada
Ejemplo 
Autoloader::add_namespace('Mynamespace', PKGPATH.'mynamespace/');

// Fíjense en las barras invertidas (backslashes), deben ser dobles si son necesarias
Autoloader::add_namespace('Nested\\Namespace', PKGPATH.'nested/');

add_namespaces($namespace, $prepend = false)

El método add_namespaces añade una matríz de espacio de nombres.

Estático Sí
Parámetros
Parámetro Por defecto Descripción
$namespace requerido una matríz de espacio de nombres
$prepend false establecer a true si quiere anteponer.
Devuelve Nada
Ejemplo 
Autoloader::add_namespaces(array(
	'Mynamespace' => PKGPATH.'mynamespace/',
	'Othernamespace' => PKGPATH.'othernamespace/',
));

namespace_path($namespace)

El método namespace_path devuelve la ruta a un espacio de nombres.

Estático Sí
Parámetros
Parámetro Por defecto Descripción
$namespace requerido the namespace
Devuelve la ruta al espacio de nobmres o false cuando no existe
Ejemplo 
// /ruta/a/mi/espacio/de/nombres
$path = Autoloader::namespace_path('Mynamespace');

// false
$path = Autoloader::namespace_path('Unknown\\Namespace');

add_class($class, $path)

El método add_class añade la ruta de una clase.

Estático Sí
Parámetros
Parámetro Por defecto Descripción
$class requerido el nombre de la clase
$path requerido la ruta hacia el archivo de la clase
Devuelve Nada
Ejemplo 
Autoloader::add_class('Namespace\\Class', PKGPATH.'namespace/classes/class.php');

add_classes($classes)

El método add_class añade una matríz de clases.

Estático Sí
Parámetros
Parámetro Por defecto Descripción
$classes requerido the array of classes
Devuelve Nada
Ejemplo 
Autoloader::add_classes(array(
	'Namespace\\Class' => PKGPATH.'namespace/classes/class.php',
	'Namespace\\Other' => PKGPATH.'namespace/classes/other.php',
));

alias_to_namespace($class, $namespace = '')

El método alias_to_namespace establece un alias a una clase a un espacio de nombres, la raíz por defecto.

Estático Sí
Parámetros
Parámetro Por defecto Descripción
$class requerido el nombre de la clase
$namespace '' el espacio de nombres
Devuelve Nada
Ejemplo 
Autoloader::alias_to_namespace('Mynamespace\\Myclass');

// Alias a un espacio de nombres
Autoloader::alias_to_namespace('Mynamespace\\Myclass', 'Other\\Namespace\\');

add_core_namespace($namespace, $prefix = false)

El método add_core_namespace añade un espacio de nombres a cuyas clases se podrían acceder sin el prefijo del espacio de nombres y que tendrán un alias hacia el espacio de nombres global. Prefijar las clases sobreescribirán clases del núcleo y espacio de nombres añadidos anteriormente.

Estático Sí
Parámetros
Parámetro Por defecto Descripción
$namespace requerido el espacio de nombres
$prefix false establecer a true para prefijar el espacio de nombres
Devuelve Nada
Ejemplo 
Autoloader::add_core_namespace('Mynamespace');

load($class)

El método load intenta encontrar a la clase, y ejecuta su función _init() si existe.

Estático Sí
Parámetros
Parámetro Por defecto Descripción
$class requerido the class name
Devuelve Nada
Ejemplo 
Autoloader::load('Model_Users');

Ayudantes procedimentales

import($path, $folder = 'classes')

La función import importa un archivo PHP dentro de COREPATH.

Parámetros
Parámetro Tipo Por defecto Descripción
$path string requerido la ruta hacia el archivo
$folder string 
'classes'
 la carpeta para prefijar la ruta, en relación con COREPATH
Devuelve string
Ejemplo 
// importar Spyc
import('spyc/spyc', 'vendor');
