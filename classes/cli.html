Clase Cli

Interactuar con la línea de comandos aceptando opciones de entrada, parámetros y texto de salida.

beep($num = 1)

El método beep hará que suene un beep del sistema en el ordenador donde se ejecute el comando.

Estático Sí
Parámetros
Parámetro Por defecto Descripción
$num 1 Número de beeps.
Devuelve nada
Ejemplo 
Cli::beep(25);

color($text, $foreground, $background = null, $format = null)

El método color cambia el color del trozo de texto.

Estático Sí
Parámetros
Parámetro Por defecto Descripción
$text requerido Cadena a ser colorida.
$foreground requerido Color principal de la cadena de texto.
$background null Color de fondo de la cadena de texto.
$format null Otro formateo a aplicar. Actualmente solo el formato 'underline' (subrayar) es soportado.
Devuelve string
Ejemplo 
if (true === false)
{
	$message = Cli::color('Error: The universe is broken.', 'red');
}

else
{
	$message = Cli::color('All is well with the world.', 'green');
}

Cli::write($message);

error($text)

El método error escribirá una línea de texto a la línea de comandos como error (parecido a write pero usa STDERR en vez de STDOUT).

Estático Sí
Parámetros
Parámetro Por defecto Descripción
$text cadena de texto vacía Texto a imprimir a STDERR para la salida.
Devuelve nada
Ejemplo 
Cli::error('Failure: You hit the wrong key with your chubby hands, try using a stick to poke the keyboard.');

prompt($question = null, $options = array())

El método prompt solicitará al usuario a que introduzca algún dato.

Estático Sí
Parámetros
Parámetro Por defecto Descripción
$question null Preguntar al usuario una pregunta y esperar a que introduzca algo.
$options array() Una matríz de opciones para el usuario las cuales podrá seleccionar.
Devuelve string
Ejemplo 
// Espera por cualquier tecla de entrada
Cli::prompt();

// Permite cualquier valor
$color = Cli::prompt('What is your favorite color?');

// Permite cualquier valor, pero ofrece un valor por defecto
$color = Cli::prompt('What is your favorite color?', 'white');

// Solo aceptará las opciones del array
$ready = Cli::prompt('Are you ready?', array('y','n'));

option($name, null)

El método option acepta una opción desde el comando inicial.

Estático Sí
Parámetros
Parámetro Por defecto Descripción
$name requerido Nombre de la opción.
$default null Un valor por defecto en caso de que ninguna opción sea introducida.
Devuelve string
Ejemplo 
$ php index.php user -v --v -name=John --name=John

wait($seconds = 0, $countdown = false)

El método wait hará que la salida de la línea de comandos espere por un número determinado de segundos establecidos y opcionalmente mostrar la cuenta atrás.

Estático Sí
Parámetros
Parámetro Por defecto Descripción
$seconds 0 Number of seconds to wait.
$countdown false Show the countdown in the output.
Devuelve nada
Ejemplo 
Cli::write('Loading...');
Cli::wait(5, true);

write($text = '', $foreground = null, $background = null)

El método write escribirá una línea de texto a la línea de comandos.

Estático Sí
Parámetros
Parámetro Por defecto Descripción
$text empty string Texto a imprimir en la salida de la línea de comandos.
$foreground null Color de la cadena de texto.
$background null Color de fondo de la cadena de texto.
Devuelve nada
Ejemplo 
Cli::write('Hello World!');

stdout($resource = null)

Cambia o devuelve el flujo de salida estándar actual. Esto es STDOUT por defecto.

Date cuenta que la propiedad pública $nocolor puede ser establecida a verdadero (true) para forzar la toda la salida a texto plano.

Estático Sí
Parámetros
Parámetro Por defecto Descripción
$resource null Cualquier manejador de archivos escribible, o nulo para devolver el manejador de archivos actual.
Devuelve El valor anterior del manejador de archivos (o el valor actual si no lo estás cambiando)
Ejemplo 
$buffer = fopen('php://temp', 'w+');
$stdout = Cli::stdout($buffer);
Cli::write("Hello World!");
Cli::error("Where's my text? :(");
Cli::stdout($stdout);
Cli::write("There it is!");

rewind($buffer);
file_put_contents('out.log', stream_get_contents($buffer));

// $ cat out.log
// Hello World!

stderr($resource = null)

Cambia o devuelve el flujo de stderr actual. Esto es STDERR por defecto.

Date cuenta que la propiedad pública $nocolor puede ser establecida a verdadero (true) para forzar la toda la salida a texto plano.

Estático Sí
Parámetros
Parámetro Por defecto Descripción
$resource null Cualquier manejador de archivos escribible, o nulo para devolver el manejador de archivos actual.
Devuelve El valor anterior del manejador de archivos (o el valor actual si no lo estás cambiando)
Ejemplo 
$errors = fopen('php://temp', 'w+');
$stderr = Cli::stderr($errors);
Cli::write("Hello World!");
Cli::error("Where's my text? :(");
Cli::stderr($stderr);
Cli::write("There it is!");

rewind($errors);
file_put_contents('out.log', stream_get_contents($errors));

// $ cat out.log
// Where's my text? :(
