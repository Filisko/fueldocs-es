Tareas

¿Qué es una tarea?

Las tares son clases que buen ser ejecutas a través de la línea de comandos o configuradas como cron job. Generalmente se usan para procesos en segundo plano, tareas programadas y tareas de mantenimiento. Las tareas pueden llamar a modelos y a otras clases tal cual como los controladores.

Creando una tarea

En Fuel las tareas se ponen en el directorio fuel/app/tasks. Más abajo hay un ejemplo de la tarea "ejemplo":

namespace Fuel\Tasks;

class Ejemplo
{

	public function run($mensaje = '¡Hola!')
	{
		echo $mensaje;
	}
}

Esta tarea será llamada a través de la utilidad refine de oil:

$ php oil refine ejemplo "¡Buenos días!"
.

Cuando se usa el mismo nombre que la tarea en la línea de comandos, el método "run()" es el que es referenciado.

Diviendo tareas en más métodos

Puedes añadir métodos a las clases de tus tareas para dividir un grupo de tareas en tareas más específicas que pueden ser llamadas por separado.

public function fecha_actual()
{
	echo date('Y-m-d');
}

Podemos llamar a este método usando:

$ php oil refine ejemplo:fecha_actual
