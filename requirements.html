Requisitos

Servidor Web
  • Cualquier servidor web. Fuel fue testeado en Apache, IIS y Nginx.
  • Funciona en todos los sistemas operativos más importantes, incluyendo *Unix y Windows./li>
Versión PHP
  • Version 5.3.3 o mayor requerida
Extensiones PHP

Fuel usa funcionalidad la cual PHP define como "extensiones". Para Fuel uses functionality that PHP defines as "extensions". Para evitar cualquier problema posible de incompatibilidad, Fuel solo usa extensiones que son parte del código PHP documentado en el sitio web de PHP. Si las extensiones usadas son parte del repositorio PECL (que necesitan ser instaladas manualmente por el administrador de sistemas), Fuel las usará si están presentes, y se las arreglará si no.

Esta es una lista de las extensiones usadas, y las clases que las usan:

Extensión Usada en Comentarios
fileinfo Clase Upload Documentación.
Tenga en cuenta que en plataformas Windows, una instalación manual de un DLL podría ser necesario.
mbstring n/a Documentación.
Si no está presente, el framework no tendrá soporte multibyte. Puedes comprobar la constante MBSTRING para ver si tu instalación tiene el soporte multibyte habilitado.
mcrypt Clase Crypt Documentación.
Si no está presente, PHPSecLib será usado para emular su funcionalidad.
PHPUnit
  • La versión 3.7 o mayor de PHPUnit es requerida si quieres ejecutar tests unitarios.

Recomendado

  • Servidor Web - Apache 2.4+, siendo ejecutado en *nix.
  • mod_rewrite - Para remover el index.php de la URL.
