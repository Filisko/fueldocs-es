Desarrolladores del núcleo

Estos son los desarrolladores que realizan commits sobre una base casi diaria, y toman todas las decisiones finales sobre el framework.

  • Harro Verton -Director del Proyecto, mantenedor del código base 1.x y el núcleo 2.0
  • Steve West - Desarrollador, jefe de desarrollo para el ORM 2.0, fieldsets y validaciones.

¿Crees que tienes lo que es necesario para convertirse en un Desarrollador del núcleo? Comienza discutiendo el futuro del proyecto en IRC, envía pull requests para problemas abiertos, ataca una de las muchas peticiones futuras, o únete al trabajo para la nueva versión 2.0.
